# hydrooj-rest-api

Read-only REST API addon for [HydroOJ](https://github.com/hydro-dev/Hydro).

[English](#english) | [简体中文](#简体中文)

---

<a name="english"></a>
## English

This package provides a RESTful interface for HydroOJ, enabling remote access to problems, submissions, and more via JWT authentication.

**Note:** This is the server-side addon. If you are looking for the command-line client, see [hydrooj-cli](https://www.npmjs.com/package/hydrooj-cli).

### Documentation
For full installation and configuration instructions, please visit the [GitHub repository](https://github.com/felixtsu/hydro-restful-api).
