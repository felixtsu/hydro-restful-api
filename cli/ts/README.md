# hydrooj-rest-cli

Command-line client for the **[hydrooj-rest-api](https://www.npmjs.com/package/hydrooj-rest-api)** HydroOJ addon (read-only `/rest-api/*` + JWT login).

```bash
npm install -g hydrooj-rest-cli
hydrooj-rest help
```

Full documentation lives in the [repository root README](https://github.com/OWNER/hydrooj_rest_api).

Replace `OWNER` with your GitHub org/user after you publish.
