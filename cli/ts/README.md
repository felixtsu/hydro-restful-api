# hydrooj-cli

CLI client for the [hydrooj-rest-api](https://www.npmjs.com/package/hydrooj-rest-api) HydroOJ addon.

[English](#english) | [简体中文](#简体中文)

<a name="english"></a>
## English

A Node.js command-line tool to interact with HydroOJ instances that have the REST API addon installed. Supports viewing problems, submissions, and more.

### Documentation
For full usage instructions and commands, please visit the [GitHub repository](https://github.com/felixtsu/hydro-restful-api).
